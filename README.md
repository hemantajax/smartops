# SmartOps AI – Enterprise Operations Dashboard

AI-augmented enterprise dashboard for managing business data with an intelligent assistant.

## Tech Stack

| Layer    | Technology                          |
| -------- | ----------------------------------- |
| Frontend | React 19 + Vite + Tailwind + shadcn |
| Backend  | NestJS (TypeScript)                 |
| Database | PostgreSQL (Supabase)               |

## Project Structure

```
smartops-ai/
├── frontend/   # React 19 + Vite app
├── backend/    # NestJS API
└── SPEC.md     # Architecture spec
```

## Quick Start

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run start:dev
```
